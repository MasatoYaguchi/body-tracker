# 体重・体脂肪率管理アプリ - プロジェクト概要

## 📋 プロジェクト概要

**React 19 + Hono + Neon PostgreSQL + Drizzle ORM**で構築された体重・体脂肪率管理アプリケーション

### アプリケーションの目的
- 個人の体重・体脂肪率データの記録・管理
- 統計情報の可視化（体重変化、体脂肪率変化）
- 友人・知人での早期利用を目指した実用的なWebアプリ
- フロントエンド開発者のバックエンド学習プロジェクト

### 主要機能
- **データ管理**: 体重・体脂肪率の記録・編集・削除
- **統計表示**: 記録数、最新値、前回からの変化量
- **認証システム**: Google OAuth認証（実装予定）
- **レスポンシブUI**: モダンなデザインで各デバイス対応

### React Hooks最適化
#### useCallbackの依存関係管理
- **基本原則**: 依存配列に使用する全ての値を含める
- **ESLintルール**: `exhaustive-deps`で依存関係を厳密チェック
- **最適化タイミング**: 子コンポーネントのpropsや重い処理でのみ使用
- **依存関係の注意点**: オブジェクト・配列は参照比較のため、useMemoと併用
- **startTransition除外**: `startTransition`は依存関係に含めない（React内部で安定している）

```typescript
// 適切な依存関係の例
const handleSubmit = useCallback((data: WeightData) => {
  startTransition(() => {
    onSubmit(data, userId);
  });
}, [onSubmit, userId]); // startTransitionは依存関係に含めない
```
